03. To Title Text
  Ala bala a KEKS -> Ala Bala A Keks
  Ala bala a!KEKS -> Ala Bala A!keks

  with punctuation
  Ala,bala a!KEKS -> Ala Bala A!Keks

04. Replace string in a string
04.1 - same length
  text: al ala bala a keks al
  replace: ala
  with: xxx
  -> al xxx bxxx a keks al

04.2 - same length or shorter
  text: ala bala a keks
  replace: ala
  with: XX
  XX bXX a keks


04.3 - new text is longer
  text: ala bala a keks
  replace: ala
  with: XXXX
  XXXX bXXXX a keks
